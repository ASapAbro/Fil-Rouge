# Carnet de Contacts

## Installation

### Backend
1. `cd backend`
2. `npm install`
3. `node server.js`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm start`

## Fonctionnalités
- Authentification JWT
- CRUD complet des contacts
- Interface React moderne
- Documentation Swagger intégrée

## API
- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/contacts`
- `POST /api/contacts`
- `PUT /api/contacts/:id`
- `DELETE /api/contacts/:id`
